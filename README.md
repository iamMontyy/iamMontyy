## 👨‍💻 Tentang Saya

Saat ini saya adalah mahasiswa Sistem Informasi di Universitas Indraprasta PGRI (UNINDRA). Saya memiliki minat yang mendalam pada pengembangan web *full-stack* serta eksplorasi teknologi *Artificial Intelligence* dan *Computer Vision* untuk menciptakan solusi praktis. Selain *coding*, saya juga aktif dalam organisasi sosial dan kepemimpinan komunitas.

* 🔭 **Saat ini sedang mengerjakan:** Sistem Penerimaan Mahasiswa Baru (PMB) digital dan proyek AI **Tracking Auto Slide Hands Presentation** menggunakan Python.
* 🌱 **Fokus pembelajaran:** Node.js lanjutan, integrasi API, dan implementasi *Computer Vision* (OpenCV & MediaPipe).
* 🤝 **Terbuka untuk kolaborasi pada:** Proyek *open-source*, pengembangan aplikasi web skala menengah, dan inisiatif teknologi berdampak sosial.
* ⚡ **Fakta menarik:** Saya percaya teknologi terbaik adalah yang terasa senatural mungkin bagi penggunanya.
